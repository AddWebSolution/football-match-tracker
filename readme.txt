Football Match Tracker is a simple and easy wordpress plugin used to show live football  score, competition list, match details, fixtures etc.

== Description ==
Football Match Tracker is a simple plugin used for show live score of football, matches list, match details, competitions list, competition details, teams list according to competition, team details, players' list according to team and player's details. User needs api authentication key from www.football-api.com for access data of api.

When You activate this plugin it will create two pages for shortcodes, you can edit pages or you can access shortcode into your posts.

Plugin provides you two widget for score and commentary.

You can change color setting of shortcodes and widgets according to your website from plugin setting menu.



== Installation ==

**Installation Instruction & Configuration**    

1. Download the zip file and extract the contents. Upload the 'football-api' folder to your plugins directory (wp-content/plugins/).

2. Activate the plugin through the 'Plugins' menu in WordPress.  

3. Go to Dashboard -> Football Match Tracker Settings and provide your valid authentication key (Get form www.football-api.com) and save changes now you can access api data. You can also change theme for shortcodes and widgets.
